================================================================================
## CRITICAL RULES
================================================================================

### RULE 1: Every Component MUST Have Complete Pins Array
- Every component MUST have a `pins` array
- Pin numbers must be strings ("1", "2", not 1, 2)
- Each pin must have: number, name, type

### RULE 2: Use Standard Reference Designators
- Resistors: R1, R2, R3, ...
- Capacitors: C1, C2, C3, ...
- Inductors: L1, L2, L3, ...
- Diodes/LEDs: D1, D2, ...
- Transistors: Q1, Q2, ...
- ICs: U1, U2, ...
- Connectors: J1, J2, ...

### RULE 3: Include Bypass/Decoupling Capacitors
- Every IC power pin needs a 100nF ceramic bypass capacitor nearby
- Add bulk capacitors (10uF+) for power supply filtering
- Add input capacitors for voltage regulators

### RULE 4: MANDATORY VOLTAGE DERATING (VIOLATION = CIRCUIT FAILURE)
These are HARD CONSTRAINTS, not guidelines. Violating them will destroy components.
All voltage-rated components MUST be derated to ensure reliability:

**MOSFETs/Transistors**: Vds ≥ 2x operating voltage
- System at 48V → Select ≥100V MOSFETs
- System at 200V → Select ≥500V MOSFETs
- System at 360V → Select ≥800V MOSFETs

**Diodes**: Vrrm ≥ 2x operating voltage
- System at 48V → Select ≥100V diodes
- System at 200V → Select ≥500V diodes
- System at 360V → Select ≥800V diodes
- AVOID: 1N4148 (100V) in high-voltage circuits
- AVOID: SS34 (40V) in high-voltage circuits
- USE: 1N4007 (1000V), UF4007 (1000V), ES1J (600V), STTH112 (1200V) for high-voltage

**Capacitors**: Voltage rating ≥ 1.5x operating voltage
- 48V rail → Select ≥100V capacitors
- 200V rail → Select ≥400V capacitors

**Resistors**: Power rating ≥ 2x expected dissipation
- Calculate P = V²/R or P = I²R
- If P = 0.5W → Select ≥1W resistor

**Inductors**: Saturation current ≥ 1.5x peak current
- Peak current 5A → Select ≥7.5A saturation inductors

### RULE 5: MICROCONTROLLER PIN LIST RULE
When selecting a microcontroller or FPGA, define ALL pins including every VCC/VDD,
GND/VSS, GPIO, peripheral, and communication pin. Mark unused pins as NC.
The Connection Agent CANNOT use any pin you do not define here.

### RULE 6: CLOCK SOURCE RULE
Every DDS, PLL, or frequency synthesizer component REQUIRES an explicit clock/reference
source component (crystal oscillator, TCXO, MEMS oscillator, clock generator IC).
Do NOT assume an internal clock unless the datasheet explicitly guarantees one.

### RULE 7: Select Real, Commonly Available Parts
- Use generic part values where possible (10k, 100nF, etc.)
- Use standard packages (0603, 0805, TO-220, SOIC-8, etc.)
- Avoid obsolete or hard-to-find parts

================================================================================
## MANDATORY PRE-OUTPUT VALIDATION
================================================================================

Before outputting your component list, verify ALL of the following:
1. Every IC has VCC/VDD and GND/VSS pins in its pin list
2. Every MOSFET voltage rating >= 1.5x module operating voltage
3. Every electrolytic capacitor voltage rating >= 1.5x module operating voltage
4. No two components share the same reference designator
If any check fails, fix the issue before outputting.

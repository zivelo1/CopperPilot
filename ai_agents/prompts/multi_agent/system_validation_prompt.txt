================================================================================
## SYSTEM-LEVEL CHECKS
================================================================================

### CHECK 1: Power Integrity
- Every IC has power connection
- Power rails have adequate decoupling
- No power rail has < 2 connections
- Power source module provides all required rails

### CHECK 2: Ground Integrity
- Single ground reference (or intentionally split grounds)
- Ground connects to all modules
- Ground has high connectivity (most connected net)

### CHECK 3: Interface Completion
- All interface signals from contracts are present
- Output signals have corresponding inputs
- No orphaned interface signals

### CHECK 4: Module Connectivity
- Every module has at least one connection to another module
- No orphaned modules (floating modules)
- Inter-module signals properly routed

### CHECK 5: No Critical ERC Violations
- No shorted passives
- No ICs without power/ground
- No floating critical pins

### CHECK 6: Component Consistency
- Reference designators are unique
- No duplicate component references
- Package assignments are consistent

================================================================================
## SEVERITY LEVELS
================================================================================

- **critical**: System will NOT work. Cannot proceed.
- **error**: System may not work correctly. Should be fixed.
- **warning**: System will work but could be improved.
- **info**: Informational note for designer.

================================================================================
## SPECIAL CONSIDERATIONS
================================================================================

### High Voltage Circuits
- Check isolation between high/low voltage sections
- Verify creepage/clearance considerations noted

### High Frequency Circuits
- Check for matched impedance nets noted
- Verify bypass cap placement considerations

### Power Electronics
- Check current paths are adequate
- Verify thermal considerations noted

Return ONLY the JSON object.

================================================================================
## VALIDATION CHECKS TO PERFORM
================================================================================

### CHECK 1: IC Power Connections (CRITICAL)
Every IC must have:
- At least one pin connected to a power rail (VCC, VDD, etc.)
- At least one pin connected to ground (GND, VSS, etc.)

### CHECK 2: Shorted Passives (CRITICAL - LAW 4)
For every 2-terminal passive (R, C, L, D):
- Pin 1 MUST be on a DIFFERENT net than Pin 2
- If both pins on same net = SHORTED = circuit won't work

### CHECK 3: Floating Pins (WARNING)
All pins should be connected:
- Unconnected pins should be marked as NC
- Critical signal pins must be connected

### CHECK 4: Single-Ended Internal Nets (WARNING)
Internal nets should have at least 2 connections:
- Interface signals are OK to be single-ended (connect to other modules)
- NC nets are OK to be single-ended
- Other internal nets should connect 2+ pins

### CHECK 5: Interface Signal Exposure (WARNING)
If interface specifies an output signal:
- That signal name should appear as a net
- At least one pin should be connected to it

### CHECK 6: Power Rail Integrity (WARNING)
Power rails should have multiple connections:
- VCC should connect to ICs AND bypass capacitors
- GND should connect to ICs AND bypass capacitors

================================================================================
## SEVERITY LEVELS
================================================================================

- **critical**: Circuit will NOT work. Must be fixed.
- **error**: Circuit may not work correctly. Should be fixed.
- **warning**: Circuit will work but could be improved.
- **info**: Informational note, no action needed.

================================================================================
## ISSUE TYPES
================================================================================

- ic_missing_power: IC has no VCC/VDD connection
- ic_missing_ground: IC has no GND/VSS connection
- shorted_passive: 2-terminal component has both pins on same net
- floating_pin: Pin has no net assignment
- floating_component: Component has no connections at all
- single_ended_net: Net has only 1 connection
- missing_interface_signal: Required interface signal not present
- missing_bypass_cap: IC lacks nearby bypass capacitor

Return ONLY the JSON object.

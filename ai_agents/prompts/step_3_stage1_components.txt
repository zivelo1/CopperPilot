## CRITICAL: COMPONENT RATING SELECTION (UNIVERSAL FOR ALL CIRCUITS)

**COMPONENT RATINGS MUST MATCH APPLICATION REQUIREMENTS**

This applies to ALL circuit types - from simple LED blinkers to high-power amplifiers, from low-frequency audio to RF circuits.

### UNIVERSAL RATING RULES:

**1. MOSFET/Transistor Voltage Rating:**
- Vds rating MUST be at least 1.5x the peak voltage (2x for high voltage applications)
- For <= 24V systems: Use 40-60V rated parts
- For 25-50V systems: Use 80-100V rated parts
- For 51-100V systems: Use 150-200V rated parts
- For 101-200V systems: Use 300-400V rated parts
- For 201-400V systems: Use 500-600V rated parts
- For > 400V systems: Use 600V+ or IGBTs

**2. Capacitor Voltage Rating:**
- Rating MUST be at least 1.5x working voltage (2x for electrolytics)
- ALWAYS specify voltage rating: "100uF/100V" not just "100uF"

**3. Resistor Power Rating:**
- Calculate P = I²R or P = V²/R for expected current/voltage
- Use rating at least 2x calculated power
- 0603: 0.1W, 0805: 0.125W, 1206: 0.25W, 2512: 1W

**4. Inductor Current Rating:**
- Saturation current must exceed peak current by 20%+
- DC resistance affects efficiency

**5. Diode Ratings:**
- Reverse voltage: 1.5x peak reverse voltage minimum
- Forward current: 2x average current minimum
- For rectifiers: Use appropriate recovery speed

{{ component_rating_guidance }}

### MANDATORY VOLTAGE DERATING (VIOLATION = CIRCUIT FAILURE):
These are HARD CONSTRAINTS, not guidelines. Violating them will destroy components.

- **Power MOSFETs**: Vds >= 2x the rail voltage they switch. A 400V rail requires >=800V MOSFETs.
  Do NOT select MOSFETs below this threshold regardless of familiarity with the part.
- **Diodes**: Vrrm >= 2x peak reverse voltage.
- **Capacitors on power rails**: Voltage rating >= 1.5x rail voltage (2x for electrolytics).

### MICROCONTROLLER PIN LIST RULE:
When selecting a microcontroller or FPGA, you MUST define ALL pins including:
- Every VCC/VDD and GND/VSS pin (there may be multiple)
- Every GPIO, peripheral, and communication pin used in the design
- Mark genuinely unused pins as NC (no-connect)
The Connection Agent CANNOT use any pin you do not define here.

### CLOCK SOURCE RULE:
Every DDS (Direct Digital Synthesis), PLL (Phase-Locked Loop), or frequency synthesizer
component REQUIRES an explicit clock/reference source component in the same module.
Acceptable clock sources: crystal oscillator, TCXO, MEMS oscillator, clock generator IC.
Do NOT assume an internal clock unless the datasheet explicitly guarantees one.

### AUXILIARY PIN RULE: NO NAKED IC PINS (Critical Circuit Completeness)
Every IC pin that is NOT power, ground, or a cross-module interface signal REQUIRES
at least one external passive component. ICs cannot function with floating auxiliary pins.
YOU MUST select the required passive components for ALL auxiliary pins during Stage 1.

**Common auxiliary pin categories and their required passives:**

1. **Filter/Compensation pins (FILT, FLT, COMP, MFLT):**
   Add a capacitor (typical: 100nF-10uF, consult datasheet)

2. **Offset/Calibration pins (OFS, OFSA, OFSB, TRIM):**
   Add a capacitor or resistor network (consult datasheet)

3. **Bias/Reference pins (BIAS, REF, SET, ISET):**
   Add a resistor to set operating point (consult datasheet)

4. **Soft-start/Timing pins (SS, RT, CT):**
   Add a capacitor (soft-start) or resistor (timing)

5. **Bootstrap pins (BOOT, BST):**
   Add a bootstrap capacitor (between BOOT and switching node)

6. **Average/RMS Output pins (AVG, RMS, OAVG, SUMR, SUMI):**
   Add a filter capacitor to GND for time-constant setting

7. **Buffer/Input pins (IBUF, OBUF, IHP, OHP):**
   Add a bypass capacitor to GND OR connect to signal path with filter

**RULE: For every IC in your design, review ALL its pins. If any auxiliary pin
requires an external passive per the datasheet, that passive MUST appear in
your component list. Failure to include auxiliary passives will cause single-ended
net violations in the connection stage.**

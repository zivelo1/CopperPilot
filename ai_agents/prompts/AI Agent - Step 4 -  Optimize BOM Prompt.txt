### OPTIMIZATION OBJECTIVES

1. **Supplier Consolidation**
   - Minimize number of suppliers (reduces shipping costs)
   - Each supplier order should exceed $100 for free shipping
   - Consider supplier strengths (DigiKey for ICs, LCSC for passives)

2. **Volume Optimization**
   - Identify price break opportunities
   - Standard reel quantities (e.g., 3000 for 0805 resistors)
   - MOQ considerations for better pricing

3. **Component Standardization**
   - Identify value consolidation opportunities
   - Example: Can 10.2k replace 10k resistors?
   - Reduce unique part count for inventory

4. **Risk Mitigation**
   - Ensure dual-sourcing for critical components
   - Avoid end-of-life or NRND parts
   - Consider lead times and availability

5. **Cost Reduction**
   - Identify overspecified components
   - Suggest value engineering alternatives
   - Calculate total landed cost (parts + shipping)

### ANALYSIS REQUIREMENTS

#### Price Break Analysis
For each high-quantity part, check if buying more hits a price break:
- 100 pcs: $0.10 each = $10
- 500 pcs: $0.06 each = $30 (better value if using within 1 year)

#### Standardization Opportunities
Look for:
- Similar resistor values (10k, 10.2k, 9.8k → standardize to 10k)
- Multiple ceramic cap types (consolidate to X7R where possible)
- Different packages of same value (standardize to one package)

#### Supplier Strategy
Consider:
- DigiKey: Best for ICs, fast shipping, high reliability
- Mouser: Good for brand-name components, wide selection
- LCSC: Excellent for basic passives, best prices on common parts

### CONSTRAINTS

- Maintain electrical specifications (no compromising functionality)
- Respect package types (SMD vs through-hole)
- Keep total lead time under 2 weeks
- Budget flexibility: ±20% for better long-term value

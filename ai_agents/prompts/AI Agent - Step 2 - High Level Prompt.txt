MODULE PHILOSOPHY:
- Each module = one buildable PCB.
- Combine related functions into single modules when practical.
- For multi-channel systems: each channel is ONE complete module containing all
  channel-specific functions (signal generation, amplification, matching, sensing).
- Shared infrastructure (power, controller, protection) as separate modules.

MODULE COUNT GUIDELINES:
- Simple circuits (LED driver, sensor): 2-3 modules
- Medium circuits (power supply, charger): 3-5 modules
- Complex circuits (multi-channel, RF, high-power): 4-7 modules
- Very complex systems: 6-10 modules MAX

SPECIFICATIONS:
- Every specification must be a CONCRETE number or range (e.g., "200W", "±15V", "39-49 kHz").
- No vague terms like "adequate", "sufficient", or "as needed".
- Include voltage ratings, current ratings, power ratings, frequency ranges, impedances.

CONNECTIONS:
- Every connection must specify the type (power, signal, control, feedback, sensing).
- Power connections should include current rating.
- Signal connections should include interface type (SPI, UART, analog, etc.).
